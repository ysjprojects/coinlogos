Introduction

While Coingecko API offers a robust trove of cryptocurrency data, one thing that it lacks is the ability to query cryptocurrency logos. Existing solutions are often outdated, limited in terms of scope or lacking versatility in querying methods.
Coinlogos is unique as it allows users to query cryptocurrency logos using contract address and ticker (if any), besides the conventional coin identifier used in Coingecko. The database also keeps itself up to date with new coins by querying the Coingecko API endpoint once every 24 hours.
